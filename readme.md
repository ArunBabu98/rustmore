# 🦀 Rustmore

**Rustmore** is an educational, character-level language model library built from the ground up in **Rust**. It is a direct spiritual successor and implementation of Andrej Karpathy's [makemore](https://github.com/karpathy/makemore) series.

The goal of this project is to demystify the "magic" of Autograd and Deep Learning by rebuilding the path from **Bigrams** to **Transformers**, swapping out Python/PyTorch for the type-safety and performance of **Rust** and the [Burn](https://burn.dev) framework.
